# Release Notes

## v1.2.0
- **한글 발음 토글 기능 추가** — Claude AI(Haiku)를 이용해 히라가나 발음을 한글 발음으로 변환
  - 가사 편집 화면 상단 `[ひ → 가]` 버튼으로 히라가나 ↔ 한글 발음 즉시 전환
  - 곡당 최초 1회만 API 호출, 이후 토글은 로컬 DB 캐시 사용 (추가 비용 없음)
  - Anthropic API 키는 기기에 암호화 저장 (`safeStorage`)
  - 히라가나 발음은 그대로 유지되어 학습 목적에 따라 선택 가능
