# 🎵 J-Pop 가사 번역 & 단어장

일본어 가사를 공부하기 위한 데스크톱 앱입니다.  
가사를 붙여넣으면 히라가나를 자동으로 생성해주고, 번역을 입력하며 단어를 저장할 수 있습니다.

---

## 주요 기능

- **히라가나 자동 변환** — 일본어 가사를 붙여넣으면 한자 → 히라가나 자동 변환
- **한글 발음 토글** — Claude AI를 이용해 히라가나 발음을 한글 발음으로 변환 (첫 변환 1회만 API 사용, 이후 토글은 로컬 캐시 사용)
- **가사 편집** — 원문 / 발음(히라가나 or 한글) / 번역을 한 줄씩 대응해서 보여주는 3단 구조
- **단어장** — 가사 보면서 모르는 단어를 바로 저장
  - 전체 단어장 / 노래별 단어장 보기
  - 즐겨찾기(★) 기능
  - 셔플 모드
  - 낱말카드 (플립 애니메이션, 키보드 조작)
- **노래 저장/관리** — 번역한 노래 목록을 홈 화면에서 관리

## 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| 프레임워크 | Electron 29 + React 18 + TypeScript |
| 빌드 | electron-vite |
| 히라가나 변환 | kuroshiro + kuroshiro-analyzer-kuromoji |
| 한글 발음 변환 | Claude AI API (claude-haiku, 사용자 API 키 필요) |
| 데이터 저장 | JSON 파일 (`%APPDATA%\jpop-lyrics-app\`) |
| UI | 레트로 픽셀 스타일 (PF Stardust 3.0 / Noto Sans JP) |

## 개발 환경 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (cmd 권장)
npm run dev
```

> **Windows 주의사항**  
> 시스템 환경변수에 `ELECTRON_RUN_AS_NODE=1`이 설정되어 있으면 앱이 실행되지 않습니다.  
> 시스템 환경변수에서 해당 항목을 삭제한 뒤 실행하세요.

---

## 빌드 (배포용 exe 생성)

관리자 권한 cmd에서 실행:

```bash
npm run package
```

`dist/` 폴더에 포터블 exe 파일이 생성됩니다.

## 한글 발음 기능 사용법

가사 편집 화면 우측 상단의 `[ひ → 가]` 버튼을 누르면 발음 모드를 전환할 수 있습니다.

1. **첫 번째 토글 시** — Anthropic API 키 입력 모달이 표시됩니다
   - [Anthropic Console](https://console.anthropic.com/)에서 API 키를 발급받으세요
   - 입력한 키는 기기에 암호화되어 로컬 저장되며 외부로 전송되지 않습니다
2. **변환 완료 후** — 한글 발음이 DB에 저장되어 이후 토글은 즉시 전환됩니다 (API 재호출 없음)
3. **히라가나 ↔ 한글 발음** 을 언제든지 자유롭게 전환할 수 있습니다

> **API 비용** — Claude Haiku 기준 곡당 약 $0.002 (약 3원) 수준입니다.

## 데이터 위치

저장된 노래와 단어장은 아래 경로에 JSON 파일로 저장됩니다:

```
%APPDATA%\jpop-lyrics-app\jpop-lyrics-data.json
```

다른 PC로 이전하려면 이 파일을 복사하면 됩니다.
