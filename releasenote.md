# Release Notes

## v1.3.0
- **단축키로 단어 즉시 저장** — 가사 편집 화면에서 단어를 드래그 선택 후 `Ctrl+E`를 누르면 단어장 추가 모달이 열리며 선택한 단어가 자동으로 입력됨
- **선택 힌트 팝업** — 텍스트를 선택하면 단어 근처에 `Ctrl+E로 단어 추가` 안내 팝업이 자동으로 표시됨
- **단축키 튜토리얼 토스트** — 번역 화면에 처음 진입할 때 단축키 안내 메시지가 잠깐 표시됨 (이후 재표시 안 함)
- **단어 추가 모달 개선** — `Ctrl+E` 단축키나 단어 클릭으로 모달을 열면 해당 단어가 입력 필드에 미리 채워짐

## v1.2.0
- **한글 발음 토글 기능 추가** — Claude AI(Haiku)를 이용해 히라가나 발음을 한글 발음으로 변환
  - 가사 편집 화면 상단 `[ひ → 가]` 버튼으로 히라가나 ↔ 한글 발음 즉시 전환
  - 곡당 최초 1회만 API 호출, 이후 토글은 로컬 DB 캐시 사용 (추가 비용 없음)
  - Anthropic API 키는 기기에 암호화 저장 (`safeStorage`)
  - 히라가나 발음은 그대로 유지되어 학습 목적에 따라 선택 가능
