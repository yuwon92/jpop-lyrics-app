# Release Notes

## v1.5.0
- **컬러 테마 추가** — 설정 화면에서 3가지 테마 중 선택 가능
  - 파스텔 (기본): 핑크 & 라벤더
  - 화이트: 무채색 기반의 깔끔한 라이트 테마
  - 다크: 차콜 계열의 깔끔한 다크 테마
- 선택한 테마는 앱 재시작 후에도 유지됨

## v1.4.1
- 코드 품질 개선 및 버그 수정 (내부 변경)
  - AI 응답이 비어있을 때 발생할 수 있는 크래시 수정
  - 모달 빠르게 닫을 때 React 경고 제거
  - CSS 색상 변수 체계 정비
  - 공용 `useEscapeKey` hook 추출, 중복 코드 제거

## v1.4.0
- **단어 발음 필드 추가** — 단어장에 히라가나 읽기(reading) 필드 추가, 단어 카드에 발음 표시
- **AI 자동 입력** — `Ctrl+E`로 단어 추가 시 발음(kuroshiro)과 뜻(Claude Haiku)이 자동으로 채워짐
  - 일본어가 아닌 텍스트는 자동 입력 없이 수동 입력
  - 조사 단독 선택 등 유효하지 않은 단어는 뜻 필드 비워두고 안내 힌트 표시
  - API 키가 없으면 발음만 자동 입력, 뜻은 수동 입력
- **단어 수정 기능** — 단어 카드의 ✎ 버튼으로 저장된 단어의 발음·뜻 수정 가능
- **낱말카드 발음 표시** — 낱말카드 뒷면에 히라가나 발음 추가
- **기존 단어 발음 자동 보완** — 이전에 저장된 단어(발음 필드 없음)를 불러올 때 kuroshiro로 발음을 자동으로 채워 DB에 저장

## v1.3.0
- **단축키로 단어 즉시 저장** — 가사 편집 화면에서 단어를 드래그 선택 후 `Ctrl+E`를 누르면 단어장 추가 모달이 열리며 선택한 단어가 자동으로 입력됨
- **선택 힌트 팝업** — 텍스트를 선택하면 단어 근처에 `Ctrl+E로 단어 추가` 안내 팝업이 자동으로 표시됨
- **단축키 튜토리얼 토스트** — 번역 화면에 처음 진입할 때 단축키 안내 메시지가 잠깐 표시됨 (이후 재표시 안 함)
- **단어 추가 모달 개선** — `Ctrl+E` 단축키나 단어 클릭으로 모달을 열면 해당 단어가 입력 필드에 미리 채워짐

## v1.2.0
- **한글 발음 토글 기능 추가** — Claude AI(Haiku)를 이용해 히라가나 발음을 한글 발음으로 변환
  - 가사 편집 화면 상단 `[ひ → 가]` 버튼으로 히라가나 ↔ 한글 발음 즉시 전환
  - 곡당 최초 1회만 API 호출, 이후 토글은 로컬 DB 캐시 사용 (추가 비용 없음)
  - Anthropic API 키는 기기에 암호화 저장 (`safeStorage`)
  - 히라가나 발음은 그대로 유지되어 학습 목적에 따라 선택 가능
